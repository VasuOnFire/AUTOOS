## ğŸ› Troubleshooting

### Issue: "Blueprint not found"
**Solution**: 
- Make sure you're using: `VasuOnFire/AUTOOS`
- Check you selected `render-simple.yaml`

### Issue: "Build failed"
**Solution**:
- Check the build logs
- Look for red ERROR messages (ignore warnings)
- Most common: missing dependencies (we handle this)

### Issue: "Service won't start"
**Solution**:
- Wait 5-10 minutes for first deployment
- Check if database is created first
- Look at logs for specific errors

### Issue: "Frontend shows error"
**Solution**:
- Make sure backend is deployed first
- Check backend URL in frontend settings
- Try accessing backend directly: `/docs`

### Issue: "Cold start is slow"
**This is normal!** Free tier services sleep after 15 minutes of inactivity.
- First request takes 30-60 seconds
- Subsequent requests are fast
- Upgrade to paid plan ($7/mo) to remove cold starts
